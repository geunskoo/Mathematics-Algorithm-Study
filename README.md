#  ✔ 스터디 규칙

* **매주 3문제를 풀고, 깃허브에 commit ❗**
* 매주 1회 대면 및 비대면 코드리뷰
* 난이도(중 ~ 상) 1문제 선정 후, 줌을 켠 상태로 각자 문제를 푼다.(시간제한 30분)
* 순간적으로 떠오른 머릿속 알고리즘 리뷰. 개선방향을 설정한다.
